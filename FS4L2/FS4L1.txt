+ Cấu trúc html:
    + gồm các thẻ <ten the>
    + có 2 loại thẻ:
        + thẻ mở: <ten the>
        + thẻ đóng: </ten the>
    + xuống dòng: giống như dấu cách


+ Các thẻ html cơ bản:
    + 2 loại thẻ:
        + thẻ nội dung:
            + p: paragraph
            + h: heading: ht -> h6
            + img: image
            + a: text chứa link đến 1 web khác
                + href: thuộc tính link
            + li: list
            + input
                + type = input/text/password/button/checkbox/radio, 
                value = Dang nhap, 
                placeholder = tencuaban
                + label
            + select
                + option
            + button
        
        + thẻ bao:
            + div: xác định 1 vùng trên trang web
            + header
            + footer
            + sidebar
            + span: bao 1 đoạn văn bản, ko có tác dụng về mặt ngữ nghĩa
            + ul: un-order list
            + ol: order list
            + form: chứa input, select, button
